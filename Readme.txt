Database olusturmaniza gerek yok urun eklendigi zaman database ve urunun eklendigi tablo kendisi otomatik olusur.

Rediscache icin ayarlari yaptim cache durumunu gorebilmeniz ve ekstra ayar yapmamaniz icin memorycached kullandim ama program.cs den yorum satisi acip bir redis baglanti url i verildigi taktirde redisde calisacaktir.

Result donusleri icin automapper kullanmak daha mantikli fakat proje ufak oldugu icin gereksinim duymadim.

Projeye eklenebilecek duzeltilebilecek cok nokta var fakat malum zamanimiz kisitli oldugu icin asiri uzerine dusunmedim en hizli cozume gitmeye calistim.
